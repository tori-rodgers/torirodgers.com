## üõ† Installation & Set Up

1. Clone the repository

	```sh
	git clone https://github.com/tori-rodgers/torirodgers.com.git
	cd torirodgers.com
	```

2. Install dependencies

	```sh
	npm install
	# or
	yarn
	# or
	pnpm install
	```

3. Start the development server

	```sh
	npm run dev
	# or
	yarn dev
	# or
	pnpm dev
	```
